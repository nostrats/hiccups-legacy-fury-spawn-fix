# Hiccups Legacy: Fury Spawn Fix

## RU

Hiccups Legacy: Fury Spawn Fix — это небольшой addon для Hiccups Legacy на Minecraft Forge 1.20.1.

### Назначение
Этот мод исправляет натуральный спавн:
- Night Fury
- Light Fury

Цель фикса — сделать фурий редкими, но реально встречаемыми в обычной игре.

### Текущий баланс
- Night Fury weight: 1
- Light Fury weight: 2

### Требования
- Minecraft 1.20.1
- Forge
- Hiccups Legacy 2.0.0

### Установка
1. Установить Forge 1.20.1
2. Установить Hiccups Legacy
3. Положить этот addon в папку `mods`

### Важно
- Это addon, а не замена Hiccups Legacy
- Основной мод обязателен
- Не используй сразу несколько версий fury fix одновременно

## EN

Hiccups Legacy: Fury Spawn Fix is a small addon for Hiccups Legacy on Minecraft Forge 1.20.1.

### Purpose
This mod fixes the natural spawning of:
- Night Fury
- Light Fury

The goal of this fix is to keep furies rare, but actually encounterable in normal gameplay.

### Current balance
- Night Fury weight: 1
- Light Fury weight: 2

### Requirements
- Minecraft 1.20.1
- Forge
- Hiccups Legacy

### Installation
1. Install Forge 1.20.1
2. Install Hiccups Legacy 2.0.0
3. Put this addon into the `mods` folder

### Important
- This is an addon, not a replacement for Hiccups Legacy
- The base mod is required
- Do not use multiple fury fix versions at the same time
